**REACT JS**
**WHY I CHOOSE REACT JS?**
1. React is a popular JavaScript library used to build user interfaces, especially for web applications. 
2. It was developed by Meta Platforms (formerly Facebook) to create fast and interactive web apps.
3. React uses a component-based architecture, meaning the UI is divided into reusable pieces called components. 
4. It uses a Virtual DOM, which improves performance by updating only the changed parts of the page. 
5. React is widely used with tools like Vite, Tailwind CSS, and Node.js to create modern, responsive applications.

1. Vite
2. Functional Based

4. filter function
5. virtual dom


**TAILWIND CSS**
1. Tailwind CSS is a utility-first CSS framework used to design modern and responsive websites quickly. 
2. It provides ready-made classes like `flex`, `text-center`, and `bg-blue-500` to style elements directly in HTML or React.
3. Tailwind helps developers build responsive designs easily using breakpoints like `sm`, `md`, `lg`, and `xl`.
4. It reduces the need to write custom CSS, which makes development faster and more efficient. 
5. Tailwind works well with frameworks like React, Vite to create clean and professional UI designs.
